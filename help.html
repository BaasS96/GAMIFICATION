Creating new games/game elements

Create questiongroup

The following elements are used to create a new question:

  • Gamepin: The gamepin is used to determine which game the queston belongs in.
  • Name: Used to display the short name for the questiongroup, preferably not longer than 20 characters.
  • Long name: Long name of the group.
  • Description: The description of the questiongroup.
  • Image: URL for the image used as item-avatar for the questiongroup.

Create question

The following elements are used to create a new question:

  • Gamepin: The gamepin is used to determine which game the queston belongs in.
  • Questiongroup-id: The questiongroup-id is used to determine in which question group the question should be placed.
  • Title: The title contains the name the question gets, e.g.: "Maths: multiplication".
  • Description: The description of the question. Could be used to for extra information. Supports the markup system.
  • Image: URL for the image used for the question. May be empty.
  • Question type: The type of question. Question types supported are: text, radio.
  • Question: The question to answer.
  • Answers: Possible answers to the question. Only used for questions of type radio. Enter as "question,question".
  • Right answers: Possible right answers. Enter as "question,question".
  • Use terminal: If checked, the terminal will be used.

Beautifying things

Markup system

In the marking system, most markup-tags are opened by [...] and closed by [/...]. They enclose text, which is marked up accordingly. They may be nested, but only like this: [b][i]...[/i][/b]. These tags are currently supported. An expamle of their uses is given between curly brackets.

  • [p][/p] Creates a paragraph. {[p](block of text)[/p]}
  • [n] New line. {example text.[n]}
  • [b][/b] Bold. {[b]example text[/b]}
  • [i][/i] Italic. {[i]example text[/i]}
  • [u][/u] Underlined. {[u]example text[/u]}
  • [img("URL")] Insert an image. {[img("https://www.w3schools.com/w3css/img_lights.jpg")]}
  • [a("URL")][/a] Creates a hyperlink. {[a("https://www.youtube.com/watch?v=dQw4w9WgXcQ")]example text[/a]}
