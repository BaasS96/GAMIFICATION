Creating new games/game elements

Create questiongroup

The following elements are used to create a new group of questions ("questiongroup"):

  • Gamepin: The gamepin is used to determine which game the queston belongs in.
  • Name: Used to display the short name for the questiongroup, preferably not longer than 20 characters.
  • Long name: Long name of the group.
  • Description: The description of the questiongroup. Supports the markup system. It is recommended not to use images or to create a long text. Use the description in questions for this.
  • Image: URL for the image used as item-avatar for the questiongroup.
  • Image location: The location of the image. If the image is on the internet, use the full URL (e.g.: "https://www.w3schools.com/w3css/img_lights.jpg"). Otherwise, only use the filename (e.g.: "image.jpg").

Create question

The following elements are used to create a new question:

  • Gamepin: The gamepin is used to determine which game the queston belongs in.
  • Questiongroup-id: The questiongroup-id is used to determine in which question group the question should be placed.
  • Title: The title contains the name the question gets, e.g.: "Maths: multiplication".
  • Description: The description of the question. Could be used to for extra information. Supports the markup system.
  • Image: URL for the image used for the question. May be empty.
  • Question type: The type of question. Question types supported are: text, radio.
  • Question: The question to answer.
  • Answers: Possible answers to the question. Only used for questions of type radio. Enter as "question,question".
  • Right answers: Possible right answers. Enter as "question,question".
  • Use terminal: If checked, the terminal will be used.

Beautifying things

Markup system

In the marking system, most markup-tags are opened by [...] and closed by [/...]. They enclose text, which is marked up accordingly. They may be nested, but only like this: [b][i]...[/i][/b].
Write your text in an external text editor (one you can save your text in). This editor does not save your input until you send it to the server!
These tags are currently supported. An expamle of their uses is given between curly brackets.

  • [p]...[/p] Creates a paragraph. A paragraph will create a new line before and after itself! {[p](block of text)[/p]}
  • [b]...[/b] Bold. {[b]example text[/b]}
  • [i]...[/i] Italic. {[i]example text[/i]}
  • [u]...[/u] Underlined. {[u]example text[/u]}
  • [img(URL)(alternative)(tooltip)] Inserts an image. Alternative is the text shown if the image cannot be found. The tooltip will be displayed when hovering over the image. Aforementioned tags do not support brackets (). The tags MUST be used. {[img(https://www.w3schools.com/w3css/img_lights.jpg)(Northern lights)(The Northern lights)]}
  • [a(URL)(tooltip)]...[/a] Creates a hyperlink. The tooltip will be displayed when hovering over the image. This tag MUST be used. {[a(https://www.youtube.com/watch?v=dQw4w9WgXcQ)(Click me!)]example text[/a]}
